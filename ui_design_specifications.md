# Специфікація UI/UX елементів інтерфейсу

У цьому документі детально описано стилізацію та логіку поведінки ключових елементів користувацького інтерфейсу (логотипу та кнопки додавання до кошика) в нашому e-commerce додатку. Дизайн виконано в преміальному стилі **"Luminous Slate" (Luxury Minimal)** із глибокими темними фонами, елегантними градієнтами та інтерактивними ефектами світіння (Glow).

---

## 1. Логотип бренду (ElectroLux)

Логотип є центральним елементом навігаційної панелі та відображає футуристичну естетику бренду.

### 🎨 Візуальні параметри (CSS/SCSS)
* **Текст:** `ElectroLux`
* **Шрифт:** `'Montserrat', sans-serif` (задається через змінну `$font-headings`)
* **Розмір шрифту:** `1.8rem`
* **Товщина шрифту:** `900` (Ultra-Bold)
* **Міжлітерний інтервал (Letter spacing):** `-0.03em` (щільне, преміальне накреслення)
* **Колір (Градієнт):** двоколірний лінійний градієнт під кутом 135 градусів:
  * Початковий колір (30% зміщення): `$text-color` (`#F8FAFC` — Slate 50, чистий світлий відтінок)
  * Кінцевий колір: `$primary-color` (`#3B82F6` — Blue 500, глибокий професійний синій)
  * Ефект градієнтного тексту реалізовано за допомогою властивостей:
    ```scss
    background: linear-gradient(135deg, $text-color 30%, $primary-color);
    background-clip: text;
    color: transparent;
    ```

### ✨ Ефекти інтерактивності (Hover)
При наведенні курсору миші на логотип:
1. **Ефект світіння (Glow):** з'являється м'яка синя неонова тінь за текстом:
   * `filter: drop-shadow(0 0 15px rgba(59, 130, 246, 0.4))`
2. **Мікроанімація:** логотип плавно зміщується вгору на 1 піксель:
   * `transform: translateY(-1px)`
3. **Плавність переходу:** транзиція становить `0.4s` з типом згладжування `ease`.

## 2. Кнопка «Купити» (Buy Button)

Кнопка покупки інтегрована в нижню частину картки товару (`ProductCard`) та має два виразних стани: базовий (додавання) та активний (товар уже додано).

### 🛒 Стан 1: Базовий стан (Товару немає в кошику)
* **Текст кнопки:** `КУПИТИ` (у верхньому регістрі)
* **Іконка:** `AddShoppingCart` (Material UI) — кошик зі знаком «плюс».
* **Шрифт:** `'Montserrat', sans-serif` (`$font-headings`), товщина `800`, розмір `0.85rem`, міжлітерний інтервал `0.05em`.
* **Стилі оформлення:**
  * **Фон:** напівпрозорий синій `rgba($primary-color, 0.1)` (`rgba(59, 130, 246, 0.1)`)
  * **Рамка:** тонка рамка `1px solid rgba($primary-color, 0.3)` (`rgba(59, 130, 246, 0.3)`)
  * **Колір тексту та іконки:** фірмовий синій `$primary-color` (`#3B82F6`)
  * **Скруглення кутів (Border radius):** `12px`

#### 👆 Ефекти при наведенні (Hover)
* Фон стає повністю залитим фірмовим синім кольором: `background: #3B82F6`
* Колір тексту та іконки змінюється на білий: `color: white`
* З'являється виразна тінь світіння: `box-shadow: 0 10px 20px rgba(59, 130, 246, 0.3)`
* Картка кнопки злегка збільшується для тактильного відгуку: `transform: scale(1.02)`
* Плавність переходу: `0.3s cubic-bezier(0.4, 0, 0.2, 1)`

## 3. Стан після додавання товару до кошика

Коли користувач натискає на кнопку «Купити», інтерфейс миттєво реагує для підтвердження успішної дії.

### 🟢 Зміна стану кнопки (Клас `.in-cart`)
1. **Зміна тексту:** текст змінюється на `В КОШИКУ`
2. **Зміна іконки:** замість кошика з'являється іконка `CheckCircleOutline` (кружечок з галочкою успіху).
3. **Зміна кольорової гами (Перехід на Смарагдово-Зелений):**
   * **Фон кнопки:** яскравий смарагдово-зелений колір `#10B981` (Green 500)
   * **Рамка кнопки:** колір рамки підлаштовується під новий фон — `#10B981`
   * **Колір тексту та іконки:** стає чисто білим `white`
   * **Ефект світіння (Glow):** зелене неонове підсвічування кнопки:
     * `box-shadow: 0 0 20px rgba(16, 185, 129, 0.2)`

### 🔔 Інтерактивні сповіщення (Toast)
У момент додавання товару у верхньому або нижньому кутку екрана випливає преміальне сповіщення (Toast-повідомлення) від бібліотеки `sonner`:
* **Повідомлення:** `[Назва товару] додано до кошика`
* **Стиль сповіщення:** відповідає загальній дизайн-системі додатку, має зелену іконку успіху та плавно зникає через кілька секунд.

### 💻 Довідник коду (SCSS-селектори)

```scss
// Логотип у Navbar.scss
.navbar-logo {
    a {
        font-family: $font-headings;
        font-size: 1.8rem;
        font-weight: 900;
        text-decoration: none;
        letter-spacing: -0.03em;
        background: linear-gradient(135deg, $text-color 30%, $primary-color);
        background-clip: text;
        color: transparent;
        transition: all 0.4s ease;

        &:hover {
            filter: drop-shadow(0 0 15px rgba($primary-color, 0.4));
            transform: translateY(-1px);
        }
    }
}

// Кнопка покупки в ProductCard.scss
.btn-buy {
    width: 100%;
    padding: 14px;
    border-radius: 12px;
    font-weight: 800;
    text-transform: uppercase;
    font-size: 0.85rem;
    font-family: $font-headings;
    letter-spacing: 0.05em;
    background: rgba($primary-color, 0.1);
    border: 1px solid rgba($primary-color, 0.3);
    color: $primary-color;
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
    display: flex;
    align-items: center;
    justify-content: center;
    gap: 10px;
    cursor: pointer;

    &:hover {
        background: $primary-color;
        color: white;
        box-shadow: 0 10px 20px rgba($primary-color, 0.3);
        transform: scale(1.02);
    }

    &.in-cart {
        background: #10b981;
        color: white;
        border-color: #10b981;
        box-shadow: 0 0 20px rgba(16, 185, 129, 0.2);
    }
}
```
