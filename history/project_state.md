# Текущее состояние проекта

## Технологический стек
*   **Фронтенд**: React, Vite, Sass (предупреждения компилятора скрыты в конфиге). Папка: `/client`.
*   **Бекенд**: Node.js, Express, MongoDB. Папка: `/server`.
*   **Уведомления**: библиотека `sonner` (заменила `react-toastify`).

## Последние важные изменения
*   **Vite**: Проект успешно перенесен на сборщик Vite. Точка входа — `/client/main.jsx`.
*   **Редизайн страницы Избранного (Favorites)**:
    *   Добавлен мини-профиль пользователя (имя, email, аватар).
    *   Добавлена статистика (количество товаров, общая стоимость).
    *   Реализованы массовые действия (добавить все в корзину, очистить избранное).
*   **Бекенд-логика Избранного**:
    *   Добавлен роут `DELETE /clear` в `server/routes/favorites.js` (помещен выше `/:productId` для предотвращения конфликтов).
*   **Компоненты**:
    *   Добавлена кастомная модалка `ClearFavoritesConfirmModal` для подтверждения массового удаления вместо дефолтного `window.confirm`.
